# AI Анализ Изображений и Документов

Этот проект предназначен для анализа изображений и документов с использованием искусственного интеллекта. Он включает в себя несколько скриптов для различных задач анализа.

## Структура проекта

- `analyze_image.py` - Основной скрипт для анализа изображений документов
- `analyze_product.py` - Скрипт для анализа изображений товаров
- `handwriter.py` - Скрипт для анализа рукописного текста
- `system_prompt.py` - Системные промпты для анализа документов
- `requirements.txt` - Зависимости проекта

## Установка

1. Активируйте виртуальное окружение:
   ```bash
   .venv\Scripts\activate
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

### Анализ документов
```bash
python analyze_image.py
```

### Анализ товаров
```bash
python analyze_product.py
```

### Анализ рукописного текста
```bash
python handwriter.py
```

## Конфигурация

Для работы проекта необходим файл `.env` с вашими API ключами:
```
OPENAI_API_KEY=ваш_api_ключ
```

## Зависимости

- openai - для работы с API OpenAI
- python-dotenv - для загрузки переменных окружения
- python-docx - для работы с документами Word
